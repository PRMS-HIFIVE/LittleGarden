# EscapeRoom
데브코스_최종프로젝트

### Git 컨벤션

| 커밋 타입 | 설명                          |
|-----------|------------------------------|
| feat      | 새로운 기능 추가              |
| fix       | 버그 수정                   |
| docs      | 문서 수정                   |
| style     | 코드 스타일/포맷 수정 (기능 변경 없음) |
| refactor  | 코드 리팩토링 (기능 변경 없음) |
| perf      | 성능 개선                   |
| test      | 테스트 코드 추가/수정        |
| chore     | 기타 자잘한 수정 (빌드, 설정 등)     |
| build     | 빌드 관련 파일 수정/설정 변경 |
| ci        | CI/CD 설정 변경             |
| revert    | 이전 커밋 되돌리기           |
